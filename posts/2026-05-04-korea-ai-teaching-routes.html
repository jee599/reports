한국 AX/AI/바이브코딩/LLM 강의 시장 — 1인 빌더 진입 루트

2026-05-04 · 읽기 12분 · 한국 강의 플랫폼, 1인 빌더 강사 진입, 4-에이전트 병렬 리서치

대상: 1인 AI 프로덕트 빌더 (jidonglab.com 운영 / spoonai.me·LLMTrio·AgentCrow·AgentOchester·ShortsMaker·TokenZip 보유 / 강사 경력 0). 조사 방식: 4개 서브에이전트 병렬 디스패치 — B2C 플랫폼 / B2B·공공 / 부트캠프·커뮤니티 / 트렌드·ROI. 본 회의록 직접 통합. 출처 URL 모두 보존.

핵심 6줄.
  1. 인프런이 한국 바이브코딩·Claude Code 키워드 트래픽을 사실상 독점. 셀프 업로드 가능, RS 70~90%, 첫 강의까지 7영업일 심사.
  2. 부트캠프 메인 강사는 7년+ 경력 / 대졸 이상 벽 — 1인 빌더 사실상 불가. (멋사 KDT 명시)
  3. 사용자에게 가장 빠른 데뷔 = 인프런 멘토링 + 자체 노션 강의 + X·디스콰이엇 모객. 학위·경력 검증 없이 즉시 시작.
  4. 패스트캠퍼스 B2B 생성형 AI 강사 양성캠프 = 학력·경력 무관 무료 진입로. 1인 빌더가 B2B 풀에 끼는 유일한 정공로.
  5. B2B 임원 AX 교육이 단가 천장(회당 100만~500만원+). 단 인프런 데이터 누적 후 진입.
  6. 사용자 USP = "지금 돌아가는 자체 SaaS 다수". 짐코딩(개발 워크플로우)·테디노트(엔터프라이즈 RAG)·양파(MS 출신)와 명확히 다름.

2. 상충 지점

  • 콜로소 도메인 적합도 — A1: 크리에이티브 중심 → 미스매치. A4: 에이전트 강의 가능성 언급. A1 신뢰 (직접 fetch 결과 IT/AI 비중 작음 확인).
  • 단가 추정치 격차 — A2(공공 5급 10만/시간 ~ B2B 톱티어 100만/시간) vs A4(김석현 누적 3,000시간 출강 케이스). 양쪽 모두 사실 — 등급 차이로 봐야.
  • 콜드 컨택 응답률 — A1은 패캠 PD 컨택 경로를 정상 루트로 제시, A4는 "기획서만으론 응답률 5% 미만" 경고. A4 우선 — 인프런 데이터 먼저 만들고 PD가 끌려오는 패턴이 정석.

5. 함정·리스크 (반드시 고려)

  1. 제작 시간 vs SaaS 본업 매출 침식 — 강의 1개 제작 60~120시간. 강의 출시 사이마다 본업 매출 점검 필수.
  2. 무한 강의 만들기 함정 — 강의는 본업 SaaS의 마케팅 깔때기여야 ROI가 산다. 강의 자체를 본업 삼지 말 것.
  3. PD 콜드 컨택은 인프런 데이터 없이 응답률 5% 미만. 인프런 → 패스트캠퍼스 순서 엄수.
  4. 무료 미끼는 유튜브 1~2편까지만. 본 강의는 처음부터 유료. 무료 길게 끌면 유료 전환 안 됨.
  5. "바이브코딩 강의" 시장의 사기 광고 오염 — 사용자의 신뢰 자산은 운영 중인 SaaS 메트릭 + 비용 공개. 과장 한 줄도 금지.
  6. 앤트로픽 가격 정책 리스크 — 2026년 4월 Claude Code Pro($20)→Max($100+) 이동. 강의에 비용 시뮬레이션 섹션 필수.
  7. 공공 외래강사 5급 단가(시간당 10만)에 묶이지 않기 — 시작은 가능하나 그 시장에 오래 머물면 단가 천장이 낮음.
